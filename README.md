# 회원관리 프로그램

# 사용언어
JavaFx + SeneBuilder + Oracle

# 프로그램설명
회원관리를 위해 로그인과 회원가입을 데이터베이스로 입력받아 저장학호 로그인에서 데이터베이스에있는 값과일치하면 회원정보데이터를 다른창에서 뿌려주고
아니라면 창은 그대로 멈춰있는다.
